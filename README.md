# Input Format

Input is a JSON file with the key `"graphs"`. Each graph contains nodes and weighted edges:

```json
{
  "graphs": [
    {
      "id": 1,
      "nodes": ["A", "B", "C", "D", "E"],
      "edges": [
        {"from": "A", "to": "B", "weight": 4},
        {"from": "A", "to": "C", "weight": 3}
      ]
    }
  ]
}
```

* `nodes` â€” list of vertex names (strings).
* `edges` â€” list of edges with fields `from`, `to`, and `weight` (integer).

---

# Output Format

The program generates a JSON file containing results for each graph:

* `input_stats` â€” number of vertices and edges
* `prim` and `kruskal` sections containing:

    * `mst_edges` â€” list of edges in the MST
    * `total_cost` â€” total weight
    * `operations_count` â€” number of key operations
    * `execution_time_ms` â€” runtime in milliseconds

Example output structure:

```json
{
  "results": [
    {
      "graph_id": 1,
      "input_stats": {"vertices": 5, "edges": 7},
      "prim": {...},
      "kruskal": {...}
    }
  ]
}
```

1. **Clone or download the repository:**
